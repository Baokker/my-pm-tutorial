# 第八章：产品战略

> "没有战略的产品经理，只是一个功能的搬运工。" —— 产品管理领域常见共识

## 为什么产品战略如此重要？

作为一名产品经理，你的日常工作可能被需求评审、项目排期、Bug 跟进填满。但如果你只盯着脚下的路，而不抬头看方向，就很容易陷入"忙碌但无效"的陷阱。

**产品战略**就是帮你抬头看路的能力——它决定了：

- 我们到底在为谁做产品？
- 我们要解决什么核心问题？
- 我们靠什么赢？
- 我们如何持续赚钱？

## PM 为什么需要商业思维？

很多初级 PM 会认为："商业的事情是 CEO 和业务负责人考虑的，我只要把产品做好就行。" 这种想法会严重限制你的职业发展。

看一下不同层级 PM 对商业思维的要求：

| PM 层级 | 商业思维要求 | 典型工作内容 |
|---------|-------------|-------------|
| 初级 PM（P1-P2） | 了解即可 | 理解产品的基本商业模式，能看懂竞品 |
| 中级 PM（P3-P4） | 必须掌握 | 能独立完成竞品分析，理解商业指标 |
| 高级 PM（P5-P6） | 深入理解 | 制定产品策略，负责 P&L（利润表） |
| 产品总监/VP | 战略级 | 制定产品矩阵战略，商业模式创新 |

## 商业思维的三大核心能力

### 1. 市场洞察力 —— 看清战场

- 行业趋势在哪里？
- 竞争对手在做什么？
- 用户的真实需求是什么？
- 技术变革带来了哪些机会？

### 2. 商业建模力 —— 算清账

- 产品靠什么赚钱？
- 获取一个用户要花多少钱？
- 一个用户能带来多少收入？
- 商业模式能否规模化？

### 3. 战略决策力 —— 做对选择

- 现在该做什么功能？
- 哪些市场该进，哪些该放弃？
- 产品处于哪个阶段，该用什么策略？
- 如何构建竞争壁垒？

## 学习建议

1. **带着问题学**：在学习每一节之前，先想一想你日常使用的某个产品（比如微信、抖音、淘宝），试着用学到的框架去分析它。

2. **动手练习**：每一节都有实操部分，建议你真正动手写一份分析报告，而不是只停留在"看懂了"的阶段。

3. **关注行业新闻**：产品战略离不开对行业的理解。建议关注以下信息源：
   - 36氪、虎嗅、极客公园（科技行业新闻）
   - 人人都是产品经理（PM 社区）
   - 各大公司财报（了解商业数据）

4. **建立自己的竞品库**：选择一个你感兴趣的领域，持续追踪 3-5 个产品，记录它们的变化。
