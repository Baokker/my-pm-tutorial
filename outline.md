### 第二章：核心概念与术语

#### 2.1 商业术语
| 术语 | 全称 | 含义 | 举例 |
|------|------|------|------|
| GMV | Gross Merchandise Volume | 总交易额 | 淘宝双 11 GMV |
| ROI | Return on Investment | 投资回报率 | 广告投放 ROI |
| LTV | Lifetime Value | 用户生命周期价值 | SaaS 订阅用户 |
| CAC | Customer Acquisition Cost | 获客成本 | 每获取一个用户花了多少钱 |
| ARPU | Average Revenue Per User | 每用户平均收入 | 游戏行业 ARPU |
| DAU/MAU | Daily/Monthly Active Users | 日活/月活 | 社交 App 核心指标 |
| 还有更多... |||

#### 2.2 产品术语
- MVP（最小可行产品）— 用图示说明 MVP 的理念
- PMF（Product-Market Fit）— 产品市场匹配

- 灰度发布 & 功能开关
- A/B Test 原理与流程
- 用户故事 & 验收标准

#### 2.3 技术术语（PM 需要了解的）
- 前后端分离、API、SDK

- 埋点 & 数据上报
- 性能指标（加载速度、崩溃率）
- 技术可行性评估

#### 2.4 设计术语
- UI vs UX
- 交互设计 vs 视觉设计
- 设计系统 / Design Token
- 响应式设计 & 多端适配
- 无障碍设计（Accessibility）

```
  AARRR 漏斗模型

  ┌─────────────────────────────┐
  │      Acquisition 获取       │  用户怎么找到你
  └──────────┬──────────────────┘
       ┌─────┴──────────────┐
       │  Activation 激活    │  首次体验是否好
       └──────┬─────────────┘
          ┌───┴────────────┐
          │ Retention 留存  │  用户是否回来
          └───┬────────────┘
            ┌─┴──────────┐
            │ Revenue 收入│  是否愿意付费
            └──┬─────────┘
             ┌─┴────────┐
             │Referral   │  是否推荐给别人
             │   推荐    │
             └──────────┘
```
